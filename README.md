# Bugema University - Department of Computing & Informatics

A modern, responsive website for the Department of Computing & Informatics at Bugema University. Built with pure HTML, CSS, and JavaScript, featuring a clean design with interactive elements and accessibility features.

## 🌐 Live Demo

This website is hosted on GitHub Pages and can be accessed at:
[https://[your-username].github.io/[repository-name]](https://[your-username].github.io/[repository-name])

## 📋 Features

### 🎨 **Design & UX**
- Modern, clean interface with blue color scheme
- Responsive design for desktop, tablet, and mobile
- Smooth hover effects and animations
- Professional card-based layout
- Background images for program sections

### 📱 **Pages**
- **Home** - Department overview with quick navigation
- **About** - Department information and student profiles
- **Programs** - Academic programs with expandable details
- **Faculty** - Staff profiles with photos and qualifications
- **Research** - Research areas and projects
- **News & Events** - Latest department news and announcements
- **Contact** - Contact information and inquiry form
- **Apply** - Multi-step application form with validation

### ⚡ **Interactive Features**
- Mobile-responsive hamburger menu
- Expandable program accordions with background images
- Auto-scrolling faculty profiles
- Multi-step application form with progress indicators
- Form validation and error handling
- Hover effects on cards and navigation

### ♿ **Accessibility**
- Semantic HTML5 structure
- ARIA labels and roles
- Skip-to-content links
- Keyboard navigation support
- Focus indicators
- Alt text for all images

## 🛠️ Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Modern styling with animations
- **JavaScript** - Interactive functionality
- **No frameworks** - Pure vanilla web technologies

## 🚀 Getting Started

### Local Development

1. Clone this repository:
   ```bash
   git clone https://github.com/[your-username]/[repository-name].git
   ```

2. Navigate to the project directory:
   ```bash
   cd [repository-name]
   ```

3. Open `index.html` in your web browser or use a local server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx serve .
   
   # Or simply open index.html in your browser
   ```

### GitHub Pages Setup

1. Fork or clone this repository
2. Go to repository Settings → Pages
3. Select "Deploy from a branch"
4. Choose `main` branch and `/ (root)` folder
5. Save and wait for deployment
6. Your site will be available at `https://[username].github.io/[repository-name]`
